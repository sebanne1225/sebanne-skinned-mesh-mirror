Skinned Mesh Mirror
VCC / VPM 導入手順

推奨導入方法:
VCC / VPM から導入してください。

手順:
1. VCC を開き、Settings -> Packages -> Add Repository から listing repo の index.json を追加してください。
2. VCC に次の URL を追加します
   https://sebanne1225.github.io/sebanne-listing/index.json
3. package 一覧から
   Skinned Mesh Mirror
   を追加します
4. Unity を開き、package を導入します

repo:
https://github.com/sebanne1225/sebanne-skinned-mesh-mirror
